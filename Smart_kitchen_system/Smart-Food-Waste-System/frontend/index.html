📊 Dataset

Household food waste dataset (Kaggle). Columns include family size, item type, quantity, storage duration, and waste amount. ~1,200 realistic records.

🧠 ML Models

Linear Regression predicts waste in kg. Decision Tree classifies risk (LOW / MEDIUM / HIGH). Both saved as .pkl via joblib.

⚙️ Pipeline

Missing value imputation → Label Encoding → Feature selection → 80/20 split → Training → R² + MAE evaluation → Serialised model.
